PilotManager_V2.2
---------------------

A module for phpVMS system to manage pilots in virtual airlines.

---------------------

Install:

1. Import SQL file provided into your DB in phpMyAdmin.
2. Download the repository and unzip in a desired location.
3. Upload files into coresponding folders on your website in the oreder.
4. Open your VA admin panel and navigate to "Add-Ons" section on the left side bar.
5. Click on "Pilot Manager"
